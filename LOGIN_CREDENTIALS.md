## Quick Login Instructions

2. Select "Normal User" from dropdown

5. You'll be redirected to the user dashboard

1. Go to login page
2. Select "Administrator" from dropdown
3. Enter admin credentials
4. Click "Sign In"
5. You'll be redirected to the admin dashboard

## Troubleshooting

### If Login Fails:
1. **Check Username**: Make sure you're using the exact username from the table
2. **Check User Type**: Ensure the dropdown matches the user type
3. **Check Password**: Passwords are case-sensitive
4. **Account Status**: Ensure account is not suspended

### Common Issues:
- **Wrong User Type**: Admin users must select "Administrator" dropdown
- **Typos**: Double-check spelling of username and password
- **Case Sensitivity**: Passwords are case-sensitive
